# 레시피 공유 서비스 MVP 개발 WBS (Work Breakdown Structure)

## 프로젝트 개요
- **프로젝트명**: Recipe Share (레시피 공유 서비스)
- **프로젝트 기간**: 3개월 (12주)
- **팀 구성**: 풀스택 개발자 2명 (개발자 A, 개발자 B)

---

## 기술 스택
| 영역 | 기술 |
|------|------|
| Frontend | Next.js 14, TypeScript, Tailwind CSS, Zustand |
| Backend | NestJS, TypeScript, Prisma ORM |
| Database | PostgreSQL, Redis |
| Authentication | JWT (Access + Refresh Token) |
| Storage | Local → AWS S3 (전환 가능) |
| Deployment | Docker, Kubernetes |

## 리스크 관리

## 주간 스프린트 계획

```
Week 1-2:   [기획/설계] 요구사항 → 화면/DB/API 설계
Week 3:     [환경구축] 모노레포 → Docker → CI/CD
Week 4:     [개발 시작] 인증 + 카테고리 + 공통 컴포넌트
Week 5:     [핵심 기능] 레시피 CRUD + 메인/목록 페이지
Week 6:     [핵심 기능] 이미지 업로드 + 상세/작성 페이지
Week 7:     [부가 기능] 검색/필터 + 마이페이지 + 댓글
Week 8:     [완성도] 캐싱 + 반응형 + 다크모드 + 테스트
Week 9-10:  [QA] 통합 테스트 → 성능 테스트 → 버그 수정
Week 11-12: [런칭] 운영 환경 → 베타 테스트 → 정식 오픈
```
